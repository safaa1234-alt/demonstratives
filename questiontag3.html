Learn Question Tags

What are Question Tags?

Question tags are short questions that are added to the end of a sentence. They are used to confirm or check information. For example:

You're coming to the party, aren't you?

She doesn't like pizza, does she?

In question tags:

  • If the main sentence is positive, the question tag is negative.
  • If the main sentence is negative, the question tag is positive.

How to Form Question Tags

Follow these simple rules when forming question tags:

  • If the sentence is positive: Use a negative tag. (e.g. "You are happy, aren't you?")
  • If the sentence is negative: Use a positive tag. (e.g. "She doesn't like coffee, does she?")
  • For questions about "I" use "am I". (e.g. "I am late, aren't I?")
